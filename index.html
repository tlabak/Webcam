ELECTRONICS SYSTEM DESIGN: WEBCAM

an immersive educational experience.

In this project course, one spends an entire quarter working on the development of a complete electronics embedded system. In this case, a embedded webcam.

The project comes to life utilizing programs such as: Microchip Studio (C), Eagle PCB, Brackets / Visual Studio Code (HTML/CSS/JS), and other design software.

Electronics System Design

This course is simply a "hands-on" course that allows a student to experience the full electronics system design cycle, in an accelerated manner. Tasks involved within the course include: Component Research, Soldering Practices, Microchip Studio (Firmware), Breakout Boards for Configuring and Testing the System Components, PCB Design, Website Design, and 3D Design.

The Webcam

The functioning of this embedded webcam is to stream the video data to a website and to display the streams of image data captured by the webcam. Within the "WEBCAM" page, one will find the display and start/stop of the data streaming from the webcam itself. Programming languages used: HTML, CSS, JavaScript, C
